# nagano-sky 月・惑星の昨/翌表示対応版

## 修正内容

月・惑星の昇る時刻 / 沈む時刻について、日付をまたぐケースに対応しました。

### 追加した表記

- 前日に昇る、または前日に沈む場合: `昨`
- 翌日に昇る、または翌日に沈む場合: `翌`

表示例:

```text
昇 昨23:48 / 沈 11:05
昇 13:08 / 沈 翌00:17
```

## 修正方針

これまでの処理は、当日と翌日を見ていました。
今回から以下の3日分を探索します。

```text
前日 0:00〜24:00
当日 0:00〜24:00
翌日 0:00〜24:00
```

そのうえで、

- 当日に昇る時刻がないが、当日に沈む時刻がある場合は、前日の昇る時刻を補完
- 当日に沈む時刻がないが、当日に昇る時刻がある場合は、翌日の沈む時刻を補完

という処理にしています。

## 対象

- 月の情報カード
- 惑星の情報カード

どちらも同じ `findRiseSetTimes()` と `formatRiseSetText()` を使っているため、月・惑星の両方に反映されます。
