# nagano-sky 星雲・星団候補拡張 fix版

## 変更内容

最新版 `nagano-sky-deepsky-top-priority-moon-fixed-daylabel.zip` をベースに、星雲・星団候補を増やしました。

## 候補数

- 合計: 108件
- メシエ天体: 72件
- 非メシエ天体: 36件

## 方針

- メシエ天体をさらに追加
- メシエ以外では、NGC / IC / Melotte / Stock / Collinder などを追加
- 双眼鏡や市販の小型望遠鏡で楽しみやすい対象を中心に選定
- 星図上・カード上では、従来通り「地平線上に見えている候補」から日替わりで3件のみ表示

## 追加した主な対象

### 追加メシエ天体

M10, M12, M14, M19, M23, M25, M26, M28, M29, M34, M39, M50, M52, M53, M56, M63, M64, M71, M74, M76, M77, M78, M79, M80, M93, M94, M97, M101, M106, M107, M108, M109 など。

### 非メシエ天体

Mel 25, Mel 20, Stock 2, NGC 457, NGC 7789, NGC 752, NGC 2392, NGC 4565, NGC 7000, Collinder 399, NGC 7662 など。

## 維持している仕様

- 星図上の表示は毎日ランダム3件のみ
- カードの3件と星図上のリング・ラベルは一致
- 月・惑星の昨/翌日またぎ表記
- 星雲・星団ラベル最上位優先度
- `index.html` の星雲・星団カード見出し

```html
<h2>星雲・星団<span class="cardSubTitle">～毎日ランダムで3つご紹介！～</span></h2>
```
