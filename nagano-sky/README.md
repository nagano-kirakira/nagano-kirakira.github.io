# nagano-sky 月相描画 直描き版

## ベース

`nagano-sky-moon-phase-refined-v2.zip`

## 修正内容

月アイコンの描画方式を、
**「暗い円を重ねる方式」から「明るい輪郭を直接描く方式」**
へ切り替えました。

### 変更点

- 月面の暗い円盤を下地にする
- 明るい部分を、月の縁の半円とターミネーターの楕円弧で直接描画
- 月齢7〜8〜10〜満月前の変化を、より自然に表現
- 月カードと星図の月アイコンの両方に反映

### 期待する見え方

- 月齢7前後: きれいな右半月
- 月齢8前後: 右半月ベースのまま、左側が少しふくらむ
- 月齢10以降: 左側のふくらみが大きくなる
- 満月後は左右反転した形で自然に変化
